## Installation

```bash
$ vagrant plugin install vagrant-trellis-cert
```

## Usage

```bash
# Trust all certificates on a Trellis vagrant VM
$ vagrant trellis-cert
```

## Limitations

Pull requests are welcome!

- Only works on macOS
- Certificates must at Trellis default location, i.e.: `/etc/nginx/ssl/*.`
